⚡ Guia Relâmpago da P2 — Python

Tudo que você precisa pra resolver as questões com exatidão, no estilo que a banca aceita. Decore as 10 receitas, siga o método de 4 passos e confira as resoluções comentadas. Funciona offline.

Como atacar QUALQUER questão

A P2 sempre pede a mesma coisa de jeitos diferentes: ler dados → organizar numa lista → calcular/filtrar → mostrar/gravar. Use este roteiro:

1
Leia a saída esperada PRIMEIRO. O formato do resultado te diz exatamente o que montar. Conte colunas, veja a ordem, veja o que é exibido na tela vs. gravado em arquivo.
2
Descubra o formato da entrada. Quantos campos por linha? Separador é , ou ;? Tem cabeçalho? Em dúvida, abra o .txt no Bloco de Notas (o enunciado costuma sugerir isso).
3
Escolha as receitas. Quase toda questão é a combinação de 2–3 receitas: Ler arquivo + Busca + Agrupar, ou Ler em blocos + Gravar, etc.
4
Funções separadas + BP no fim. Escreva cada função pedida (com o nome EXATO do enunciado) e, no final, o BP (Bloco Principal) que chama as funções para testar. Se o enunciado manda "use a função X", chame-a mesmo.

O esqueleto universal de uma questão de arquivo

# 1) função que LÊ e organiza numa lista
def leDados(nomeArq):
    arq = open(nomeArq, 'r')
    lista = []
    for linha in arq:
        campos = linha.strip().split(',')
        lista.append(campos)        # ou monta [campo0, int(campo1), ...]
    arq.close()
    return lista

# 2) função que EXIBE/processa
def exibe(lista):
    for item in lista:
        print(item)

# 3) BP (bloco principal) – testa tudo
dados = leDados('arquivo.txt')
exibe(dados)
Mantra: abre → for linha → strip → split → usa → close

Simulado Q1 — decodifica / agrupaCodifDecodif / exibePalavras  ·  Receitas 3, 4, 7
Decodificar: consoantes maiúsculas ficam; vogais foram trocadas pela consoante minúscula seguinte (a→b, e→f, i→j, o→p, u→v), então na volta b→A, f→E, j→I, p→O, v→U. Agrupar strings de 3 a 10 caracteres por tamanho. Exibir.
def decodifica(s):
    mapa = {'b':'A', 'f':'E', 'j':'I', 'p':'O', 'v':'U'}
    resultado = ''
    for c in s:
        if c in mapa:
            resultado = resultado + mapa[c]
        else:
            resultado = resultado + c       # consoante ou outro: não muda
    return resultado

def agrupaCodifDecodif(lista):
    grupos = []
    for i in range(8):                  # 3 a 10 letras = 8 listinhas
        grupos.append([])
    for palavra in lista:
        tam = len(palavra)
        if tam >= 3 and tam <= 10:
            nova = palavra + ':' + decodifica(palavra)
            grupos[tam - 3].append(nova)   # tam 3 → índice 0
    return grupos

def exibePalavras(grupos):
    print('Palavras agrupadas por número de letras:')
    for i in range(len(grupos)):
        if grupos[i] != []:
            print('%d letras:' % (i + 3))
            for s in grupos[i]:
                print(' ' + s)

Simulado Q2 — valor médio por hora por cidade  ·  Receitas 1, 3, 4, 8, 10
Cada linha: cidade,tipo,horas,valor. Somar horas e valores por cidade (a busca é função separada!), depois exibir cidade e valor/horas, ordenado pelo nome.
def buscaCidade(cidade, lista):
    for i in range(len(lista)):
        if lista[i][0] == cidade:
            return i
    return None

def calcular_valores_por_cidade(nomeArq):
    arq = open(nomeArq, 'r')
    resultado = []
    for linha in arq:
        campos = linha.strip().split(',')
        cidade = campos[0]
        horas  = int(campos[2])
        valor  = float(campos[3])
        ind = buscaCidade(cidade, resultado)
        if ind is None:
            resultado.append([cidade, horas, valor])      # [cidade,totHoras,totValor]
        else:
            resultado[ind][1] = resultado[ind][1] + horas
            resultado[ind][2] = resultado[ind][2] + valor
    arq.close()
    return resultado

def exibir_resultados(lista):
    lista.sort()                              # ordena pelo nome da cidade
    for item in lista:
        media = item[2] / item[1]
        print('%-20s %.2f' % (item[0], media))

Simulado Q3 — geraSenha  ·  Receita 7
Senha = 1º caractere do nome + último caractere + miolo (por faixa de idade) + mês de aniversário invertido. Ex.: "Luiz Felipe", 25, "19/04" → L+e+$%+40 = Le$%40.
def geraSenha(nome, idade, dataAniv):
    if idade < 18:
        miolo = '#*'
    elif idade < 40:
        miolo = '$%'
    else:
        miolo = '&@'
    mes = dataAniv[3:5]          # 'dd/mm' → pega 'mm'
    mesInvertido = mes[::-1]    # '04' → '40'
    return nome[0] + nome[-1] + miolo + mesInvertido

# BP – pergunta ao usuário
nome = input('Nome: ')
idade = int(input('Idade: '))
data = input('Aniversário (dd/mm): ')
print('Senha de acesso:', geraSenha(nome, idade, data))

Revisão Q4 — alocação de salas (o "monstro" da P2)  ·  Receitas 6, 2, 10
SALAS.TXT vem em blocos: predio,qtdSalas e depois as salas nomeSala,lugares. ALUNOS.TXT: nome,codTurma. Para cada aluno calcula o índice da sala ((len(nome)+parteNumérica) % qtdSalas), aloca se houver vaga (descontando 1), senão grava em SEMSALA.TXT. Não ler nenhum arquivo de uma vez só.
def leArqSalas(nomeArq):
    salas = []
    arq = open(nomeArq, 'r')
    linha = arq.readline()                       # cabeçalho do prédio
    while linha != '':                         # Receita 6: ler em blocos
        campos = linha.strip().split(',')
        qtd = int(campos[1])                   # nº de salas deste prédio
        for i in range(qtd):
            dados = arq.readline().strip().split(',')
            salas.append([dados[0], int(dados[1])])  # [nomeSala, lugares]
        linha = arq.readline()                   # próximo prédio
    arq.close()
    return salas

def determinaIndiceSalaDoAluno(qtdSalas, nome, codTurma):
    parteNum = int(codTurma[:-1])      # '98B' → 98 (tira a última letra)
    valorBase = len(nome) + parteNum
    return valorBase % qtdSalas

def processaArqAlunos(salas):
    arqAlunos = open('alunos.txt', 'r')
    arqSem = open('SEMSALA.txt', 'w')
    qtdSalas = len(salas)
    print('%-20s %s' % ('Nome', 'Sala'))
    for linha in arqAlunos:
        campos = linha.strip().split(',')
        nome = campos[0]
        cod = campos[1]
        ind = determinaIndiceSalaDoAluno(qtdSalas, nome, cod)
        if salas[ind][1] > 0:                 # tem vaga?
            print('%-20s %s' % (nome, salas[ind][0]))
            salas[ind][1] = salas[ind][1] - 1  # desconta a vaga
        else:
            arqSem.write(nome + '\n')
    arqAlunos.close()
    arqSem.close()

Revisão Q5 — temperaturas (fórmula par/ímpar)  ·  Laço + acumular, Receita 10
Dia ímpar: 3(n+2)+5(n%5−3). Dia par: 2(n+5)+4(n%12−1). Exibir a temperatura de cada dia do intervalo (com "FRIO INTENSO" se < 30) e retornar a média.
def calculaTemperatura(n):
    if n % 2 == 1:                  # ímpar
        return 3 * (n + 2) + 5 * (n % 5 - 3)
    else:                           # par
        return 2 * (n + 5) + 4 * (n % 12 - 1)

def exibeECalculaMediaTemperaturas(inicio, fim):
    soma = 0
    qtd = 0
    for n in range(inicio, fim + 1):
        t = calculaTemperatura(n)        # usa a outra função (obrigatório)
        print('Dia %d: %d°F' % (n, t))
        if t < 30:
            print('FRIO INTENSO')
        soma = soma + t
        qtd = qtd + 1
    media = soma / qtd
    print('Média das temperaturas de %d a %d: %.2f°F' % (inicio, fim, media))
    return media

Bônus — Recursão: somaAlg e exibeVert  ·  Receita 9
Recursão aparece nos slides — provável na P2. somaAlg(323)=8 (soma dos algarismos). exibeVert(875) imprime 8, 7, 5 em linhas separadas.
def somaAlg(n):
    if n < 10:                    # caso base: 1 algarismo
        return n
    return n % 10 + somaAlg(n // 10)   # último + soma do resto

def exibeVert(n):
    if n < 10:
        print(n)
        return
    exibeVert(n // 10)            # primeiro chama (imprime os de cima)
    print(n % 10)                  # depois imprime a unidade

exibeVert(875)         # 8 / 7 / 5
